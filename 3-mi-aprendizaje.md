# COMPLETAR  
Comparando sus conocimientos antes de hacer la práctica con sus conocimientos después de hacer la tarea, explicar los principales aprendizajes logrados para beneficio de su formación profesional.  
Si solucionó un problema presentado al realizar la práctica también se debe documentar.

Se fortalecieron conocimientos sobre el uso de Docker Compose para configurar y gestionar aplicaciones multi-contenedor mediante archivos YAML, considerando la definición de servicios, variables de entorno, puertos, volúmenes, redes, políticas de reinicio y healthchecks. Asimismo, se comprendió la importancia de establecer dependencias entre servicios y de utilizar comandos básicos para la ejecución y administración de contenedores en entornos como WordPress con MySQL y SonarQube con PostgreSQL.
